# LeiloesTDSat - Sistema de Gerenciamento de Leilões

## Nome do Projeto
LeiloesTDSat

## Explicação sobre o que se trata o projeto
Este é um sistema Java desenvolvido para gerenciamento de leilões de produtos. O sistema permite cadastrar produtos para leilão, gerenciar lances, visualizar produtos em leilão e acompanhar o status das vendas. Possui interface gráfica desenvolvida em Java Swing e integração com banco de dados MySQL para persistência dos dados.

## Tecnologias utilizadas
- Java (Java SE)
- MySQL (Sistema de Gerenciamento de Banco de Dados)
- JDBC (Java Database Connectivity para conexão com MySQL)
- Java Swing (Interface Gráfica do Usuário)
- Apache Ant (Ferramenta de build)
